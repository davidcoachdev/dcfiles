# Swimlane

**Best for:** cross-functional processes, handoffs between actors.

## Layout
- Lanes: horizontal (actors/functions)
- Processes: flow within lanes
- Handoffs: arrows crossing lanes
- Labels: actor name left of lane

## Anti-patterns
- Too many lanes (max 5)
- Handoffs crossing unnecessarily
- Lane labels inside diagram area
