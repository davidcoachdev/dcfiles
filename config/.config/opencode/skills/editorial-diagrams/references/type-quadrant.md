# Quadrant

**Best for:** two-axis positioning, prioritization, trade-off analysis.

## Layout
- Axes: horizontal + vertical with labels
- Quadrants: 4 regions with labels
- Items: positioned in quadrants
- Labels: above items

## Anti-patterns
- Items on axis lines
- Too many items (max 12)
- Axes without clear labels
