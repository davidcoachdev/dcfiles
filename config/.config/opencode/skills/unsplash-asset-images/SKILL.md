# Unsplash Asset Images

Guía para elegir imágenes de Unsplash con URLs curadas y sizes correctos.

## Profile

- **Uso**: Obtener imágenes de Unsplash para avatars, headshots, portraits, backgrounds, wallpapers
- **Fuente**: https://unsplash.com
- **Inspiración**: MengTo/Skills — unsplash-asset-images

## Regla de entrega

Para cada recomendación, devolver:
1. **URL de la página de Unsplash** (canonical)
2. **Ratios + tamaños sugeridos** para el caso de uso

## Licencia

- Las imágenes de Unsplash son generalmente gratis
- Evitar imágenes **Unsplash+** a menos que el user las pida explícitamente
- No es necesario atribución obligatoria, pero es buena práctica

## Cheatsheet de Ratios

| Uso | Ratio |
|-----|-------|
| Avatar | 1:1 |
| Headshot card | 4:5 (default), 3:4 (más alto) |
| Website hero | 16:9 |
| Mobile wallpaper | 9:16 |

## Tips

- Preferir espacio negativo limpio
- Evitar backgrounds ocupados
- No cortar rostros awkwardly
- Para "mejor imagen": fondo simple, rostro centrado
