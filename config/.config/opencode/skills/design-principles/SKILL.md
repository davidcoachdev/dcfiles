> **Absorbed:** `ui-design-tips` (16 Adham Dannaway tips), `ui-ux-pro` (67 styles + 161 industry rules + color palettes)

## When to Use

- Building new UI components or pages
- Creating dashboards, admin panels, SaaS tools
- Improving existing interface designs
- Adding motion, animations, or keyframe effects
- Need systematic spacing/depth/color/motion decisions
- Want design decisions remembered across sessions
