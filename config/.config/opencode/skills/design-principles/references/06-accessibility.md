# Accessibility: Contrast & Color

**Tips 7-9**: Contrast ratios and color independence.

## Contrast Ratios (WCAG 2.1 AA)

| Element | Minimum Ratio |
|---------|--------------|
| UI elements (buttons, icons, form fields) | **3:1** |
| Small text (≤18px regular) | **4.5:1** |
| Large text (≥24px regular or ≥18px bold) | **3:1** |

## Color Independence

- 8% of men have some form of color blindness
- Common: difficulty distinguishing red/green
- Use additional cues: underlines for links, icons, text labels
- **If color is removed, interface must still be understandable**

## Tools

- [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)
- [WCAG 2.1 Quick Reference](https://www.w3.org/WAI/WCAG21/quickref/)

## Checklist

- [ ] All UI elements have ≥3:1 contrast against background
- [ ] All text has ≥4.5:1 contrast (≥3:1 for large text)
- [ ] Links have underlines or other non-color indicators
- [ ] Status/meaning uses icon + color, not color alone
- [ ] Interface understandable in grayscale
