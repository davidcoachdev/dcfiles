# Space & Grouping

**Tip 1**: Use space to group related elements.

## Rules

- Place related elements in the same container
- Space related elements close together
- Make related elements look similar
- Align related elements in a continuous line
- **Containers are strongest visual cue** but add clutter — use subtle methods first

## Application

```
GOOD:  [Label     ]    BAD:  [Label]
       [Input     ]          [Input]
       [Hint text ]

       ↑ related items       ↑ floating items
         grouped close         scattered
         (8px gap)             (16px+ gaps)
```

## Priority order for grouping (subtle → strong)

1. Proximity (space alone)
2. Alignment (continuous line)
3. Similarity (same visual treatment)
4. Container (box/background)

Containers are strongest but add visual clutter. Try proximity and alignment first before wrapping everything in boxes.
