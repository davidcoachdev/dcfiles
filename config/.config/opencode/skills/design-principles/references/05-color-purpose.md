# Color With Purpose

**Tip 6**: Use color purposefully.

## Rules

- Start with black and white — add color only where it conveys meaning
- Apply brand color to interactive elements (links, buttons)
- Avoid brand color on non-interactive elements
- Color should teach what's interactive vs. not

## Color Roles

| Role | Usage |
|------|-------|
| **Primary** | Interactive elements only (buttons, links) |
| **Secondary** | Supporting interactive elements |
| **Danger** | Destructive actions |
| **Success** | Positive confirmations |
| **Neutral** | Everything else (text, borders, surfaces) |

## Anti-Patterns

- ❌ Brand color on headings
- ❌ Brand color on decorative shapes
- ❌ Colored backgrounds without meaning
- ❌ Multiple competing accent colors
