# Remove Clutter

**Tip 5**: Remove unnecessary styles.

## Rules

- Avoid unnecessary lines, colors, backgrounds, animations
- Unnecessary visual complexity increases cognitive load
- Remove decorative elements that don't convey meaning or group information

## What to Remove

- Decorative dividers that don't separate meaningful groups
- Background colors on elements that already have sufficient grouping
- Animations that don't provide feedback or guide attention
- Borders that don't establish hierarchy or separation
- Shadows used purely for decoration (use for elevation only)

## Before Adding Style, Ask

1. Does this convey meaning?
2. Does this group information?
3. Does this guide attention?

If no to all three → don't add it.
