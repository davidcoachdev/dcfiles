## Content Strategy Framework

### Content Pillar Model
```
Pillar: Topic you want to be known for
├── Hub Content: In-depth guides, cornerstone pages (1k-3k words)
├── Spoke Content: Blog posts, tutorials, case studies (500-1k words)
├── Micro Content: Social posts, tips, threads, infographics
└── Evergreen vs Timely Balance:
    - 70% evergreen (search-optimized, long-term traffic)
    - 30% timely/trending (newsjack, memes, current events)
```

### Editorial Calendar Template
```markdown
| Week | Content Type | Topic | Primary Channel | CTA | Status |

| W1   | Spoke       | [topic] | Twitter/X | Share | Idea |

```

## Platform-Specific Strategies

### Twitter/X
- **Format**: Threads (10-15 tweets max), single hot takes, polls
- **Frequency**: 3-5 posts/day, 1-2 threads/week
- **Best times**: 8-10am, 12-1pm, 5-6pm (audience timezone)
- **Growth**: Quote-tweet with added value, engage with big accounts in niche
- **Hook**: First tweet must stop the scroll (data point, bold claim, question)

### LinkedIn
- **Format**: Long-form posts (1,300 chars), carousel docs, articles
- **Frequency**: 1 post/day, 2-3 carousels/week
- **Best times**: Tue-Thu 7-8am, 12pm, 5-6pm
- **Growth**: Comment on 10 relevant posts before publishing, use storytelling framework
- **Hook**: Opening line must create curiosity gap ("I spent 3 years building X. Here's what nobody tells you about Y")"

### Instagram
- **Format**: Reels (15-30s), carousels (5-7 slides), stories (behind-scenes)
- **Frequency**: 1 reel/day, 3 carousels/week, daily stories
- **Best times**: 11am-1pm, 7-9pm
- **Growth**: Use trending audio, collaborate with micro-influencers, respond to every comment

### Reddit
- **Format**: Long-form text posts, case studies, AMAs
- **Rule**: Provide 10x value before ANY self-promotion
- **Subreddits**: Find niche communities (not r/programming for dev tools)
- **Growth**: Answer questions thoroughly, share failures transparently, NO stealth marketing

## Copywriting Formulas

### PAS (Problem-Agate-Solution)
"Struggling with X? It gets worse — Y happens. Here's how to fix it: Z."

### AIDA (Attention-Interest-Desire-Action)
"Bold hook → Fascinating detail → Emotional benefit → Clear CTA"

### BAB (Before-After-Bridge)
"Before: Painful state. After: Ideal state. Bridge: How to get there."

### 4Us (Useful-Urgent-Unique-Ultra-specific)
"5 ways to [achieve X] in [timeframe] — #3 saved me [specific result]"

## Content Optimization Checklist

- [ ] Headline includes primary keyword + emotional trigger
- [ ] Meta description under 155 chars with keyword + CTA
- [ ] Opening paragraph hooks reader within first 2 sentences
- [ ] Skimmable: H2/H3 hierarchy, bullet points, bold key terms
- [ ] Internal links: 3-5 contextual links per 1,000 words
- [ ] External links: 1-2 authoritative sources
- [ ] CTA at end: subscribe, share, comment, or next step
- [ ] Social sharing buttons or pre-written share text
- [ ] Featured image with alt text
- [ ] Schema markup for rich results

## Viral Coefficient Formula

```
K = i × c
where:
  i = number of invitations sent per user
  c = conversion rate of each invitation

K > 1 = viral growth (each user brings >1 new user)
K < 1 = linear growth (need other channels to scale)

Example:
  i = 5 (each user shares to 5 friends)
  c = 0.3 (30% of invites convert)
  K = 5 × 0.3 = 1.5 → VIRAL ✅

Levers to increase K:
  - Increase i: make sharing easier, incentivize referrals
  - Increase c: improve onboarding, reduce friction, add social proof
```
