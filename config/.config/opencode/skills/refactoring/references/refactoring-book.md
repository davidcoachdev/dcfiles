## Book

**Title:** Refactoring: Improving the Design of Existing Code
**Author:** Martin Fowler
**Original:** https://martinfowler.com/books/refactoring.html
**Website:** https://refactoring.guru/es/refactoring/book

## About

El libro clásico de refactoring (primera edición 1999, segunda 2018).
La segunda edición usa JavaScript como lenguaje principal.

## Contenido

### Primera Parte

1. **Refactoring Example** - Ejemplo completo paso a paso
2. **Principles of Refactoring** - Cuándo y cómo refactorizar
3. **Bad Smells in Code** - Los 24 code smells originales
4. **Building Tests** - Importancia de tests en refactoring

### Segunda Parte (Catálogo)

- Organizing Methods
- Moving Features Between Objects
- Organizing Data
- Simplifying Conditionals
- Making Methods Easier to Call
- Dealing with Generalization
- Big Refactorings
