# Grid — Card UI Design

## Rules

- **Use larger paddings** in a grid layout
- **Mix paddings** within same design — e.g., combine 30px + 40px to avoid monotony

## Why

Uniform padding everywhere feels rigid. Mixing values slightly creates visual interest while maintaining harmony.
