# Rule of Threes — Card UI Design

## 1. Composition

- Divide design in a 3x3 grid
- 4 intersection points are "golden spots" where eyes land naturally
- Place key content on these intersections

### Attention Distribution

| Point | Attention |
|-------|-----------|
| Top-left | 41% |
| Bottom-left | 25% |
| Top-right | 20% |
| Bottom-right | 14% |

## 2. White Space

- Leave white space in **at least 3 areas** of the design
- Allows elements to breathe, stay focused, avoid cluttered look

## 3. Typography

- Maximum **3 fonts** total:
  1. Font for headings
  2. Font for subheadings
  3. Font for body text
- More than 3 = clutter and confusion

## 4. Color Palette

- Use **60-30-10 rule**:
  - 60% dominant color
  - 30% secondary color
  - 10% accent color
- Keeps palette clean and balanced
