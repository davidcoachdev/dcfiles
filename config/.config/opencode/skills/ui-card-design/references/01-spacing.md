# Side Spacing — Card UI Design

## Rules

- **Side spacing always generous** — don't squeeze content against edges
- **Top spacing slightly larger** than side spacing → activates top area, draws subtle attention upward

## Example

```
Card:
  padding-left: 24px
  padding-right: 24px
  padding-top: 28px     ← slightly larger
  padding-bottom: 24px
```

## Why

Extra top spacing creates visual breathing room. User eye enters from top — more space there = more interesting composition.
