# Balance & Tension — Card UI Design

## Rules

- **Achieve dynamism** by balancing whitespace as a design element, not just empty space
- **Weight perception:** elements placed lower feel heavier visually
- Use this intentionally — bottom-heavy cards feel grounded, top-heavy feel light

## Why

Whitespace is not "empty" — it's an active element. Balance it like any other design component.
