# Proximity — Card UI Design

## Rules

- **Gestalt Law of Proximity:** objects close together appear to form groups, even if shapes/sizes differ
- **Create element groups** — humans naturally perceive organized patterns
- **Add new elements to create groups** when needed
- **Use avatars** when a user is identified

## Why

Grouping through proximity is stronger than visual similarity alone. Users naturally cluster nearby elements.
