# Type Contrast — Card UI Design

## Rules

- **Size hierarchy:** always need large + medium + small text
- **Depth:** size contrast creates visual depth (foreground, midground, background)

## Example

```
Card Title:    18px bold      → foreground
Card Subtitle: 14px regular   → midground  
Card Meta:     12px light     → background
```

## Why

Size contrast isn't just about readability — it creates visual layers that guide the eye through the card.
