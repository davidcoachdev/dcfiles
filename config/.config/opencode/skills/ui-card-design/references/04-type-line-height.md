# Type Line Height — Card UI Design

## Rules

- **Extra attention to line height** — know recommended font size + line height tables
- **Density:** tighter lines = heavier text feels visually
- **Visual trick:** apply Gaussian blur or squint eyes to evaluate visual weight of text block

## Recommended Line Heights

| Font Size | Line Height |
|-----------|-------------|
| 12px | 1.5-1.6 |
| 14px | 1.4-1.5 |
| 16px | 1.5-1.6 |
| 18px | 1.5-1.7 |
| 24px+ | 1.3-1.4 |
