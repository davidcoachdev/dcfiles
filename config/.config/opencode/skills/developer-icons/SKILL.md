## Cómo usar

### 1. NPM (React/Next.js)

```bash
npm i developer-icons
```

```tsx
import { HtmlIcon, ReactIcon } from "developer-icons";

<HtmlIcon size={24} color="red" />
```

### 2. Descargar SVG

Ir a https://xandemon.github.io/developer-icons/icons/All
