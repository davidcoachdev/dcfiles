## Uso

El agent carga automáticamente los skills relevantes cuando detects código Go. Las descripciones (~100 tokens) están en memoria; el SKILL.md completo se carga cuando triggers.

## Trigger

- Código Go en el workspace
- Necesitás help con testing, security, patterns
- Refactor de código Go
