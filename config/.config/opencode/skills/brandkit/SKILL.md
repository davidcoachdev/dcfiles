# ImageGen: Brand Kit

## OUTPUT
This skill generates **IMAGES only** (no code). Use with ChatGPT Images, Codex image mode, or any image-generating agent.

## PROMPT TEMPLATE

When using this skill, prompt the image generator:

```
Create a brand identity kit for [company/project]:

Logo Options:
- [Wordmark / Lettermark / Pictorial / Abstract / Mascot]
- Style: [Minimal / Bold / Playful / Premium / Technical]

Color Palette:
- Primary: [color]
- Secondary: [color]
- Accent: [color]
- Neutral: [colors]

Typography:
- Headlines: [font choice]
- Body: [font choice]
- Monospace (optional): [font choice]

Applications:
- Business card
- Website mockup
- App icon
- Social media
- Merchandise
```

## EXAMPLE PROMPTS

### Tech Startup
```
Brand kit for a developer tools company. Modern, technical feel.
Logo: geometric mark with "Dev" wordmark. Colors: Deep navy, 
electric blue accent, white. Typography: Geist for headings,
Inter for body. Show applications on website, business card,
and app icon. Clean, premium, NOT generic.
```

### Creative Agency
```
Brand identity for a creative design agency. Bold, artistic.
Logo: Abstract symbol that can animate. Colors: Black, warm 
orange accent, cream background. Typography: Large editorial
fonts for headlines, clean sans for body. Show on poster,
website, and packaging. High-end, distinctive.
```

### E-commerce Brand
```
Brand kit for a sustainable fashion brand. Organic, premium.
Logo: Minimal wordmark with leaf icon. Colors: Earth tones
(olive, terracotta, cream). Typography: Elegant serif for
headlines, readable sans for body. Show on shopping bag,
website header, and hang tag. Natural, expensive feel.
```

## DESIGN REQUIREMENTS

### Logo Directions
Show 2-3 variations:
- Primary logo
- Alternative (different orientation/size)
- Monochrome for print

### Color Palette
- Primary color (main brand)
- Secondary color (support)
- Accent color (calls to action)
- Neutral palette (grays)
- Dark mode variant (if applicable)

### Typography
- Display/Headline font
- Body font
- Weights shown (regular, bold)

### Applications
Show brand in context:
- Business card
- Website hero
- App icon
- Social post
- Packaging or merchandise

## USE CASES

- New brand creation
- Rebrand projects
- Design system foundation
- Client presentations

## FORBIDDEN

- ❌ Generic logo concepts (lightbulb, gear, globe)
- ❌ Random color combinations
- ❌ Mismatched typography styles
- ❌ No context (show in use!)
