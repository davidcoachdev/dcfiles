# Aura Asset Images

Guía para buscar y usar imágenes de stock desde Aura Assets (aura.build/assets).

## Profile

- **Uso**: Obtener imágenes de alta calidad para mockups de diseño, backgrounds, wallpapers, portraits, headshots
- **Fuente**: https://www.aura.build/assets
- **Inspiración**: MengTo/Skills — aura-asset-images

## Cómo buscar

1. Abrir https://www.aura.build/assets
2. Usar el buscador o query param: `https://www.aura.build/assets?q=<tag>&order=popular`
3. Tags que funcionan bien: `background`, `abstract`, `architecture`, `portrait`, `headshot`

## Formato de URLs

```
https://hoirqrkdgbmvpwutwuwj.supabase.co/storage/v1/object/public/assets/assets/<UUID>_800w.jpg
```

### Resolución

- `_800w` → thumbnail
- `_1600w` → alta resolución (recomendado)
- `_3840w` → 4K

Si una variante `_1600w` da 404, usar `_800w`.

## Ratios

| Uso | Ratio |
|-----|-------|
| Avatars | 1:1 (square) |
| Headshots | 4:5 o 3:4 |
| Website heroes / backgrounds | 16:9 |
| Mobile wallpapers / stories | 9:16 |

## Crop Tips

- **Faces**: mantener ojos ~1/3 desde arriba; no cortar mentón/frente
- **Backgrounds**: preservar línea del horizonte, dejar 30-50% espacio negativo para texto
- **Avatars**: centrar rostro, fondo simple
